\# Distribution Analysis Dashboard



\## 📌 Project Overview

This project focuses on performing an Exploratory Data Analysis (EDA) and building a data visualization dashboard to analyze demographic distributions, education levels, and income groups within a population dataset of approximately \*\*48.8K individuals\*\*.



\## 📊 Key Insights from the Dashboard

\* \*\*Population Split\*\*: The dataset comprises \*\*33K males\*\* and \*\*16K females\*\*.

\* \*\*Education Distribution\*\*: The majority of individuals have a High School graduation (`HS-grad`) or some college background.

\* \*\*Age Metrics\*\*: The population shows a steady distribution across early adulthood to middle age, with peak counts concentrated between ages 23 and 37.

\* \*\*Income Groups\*\*: A significant majority (\*\*76.0%\*\*) of the individuals earn `<=50K`, while the remaining segment earns `>50K`.



\## 🛠️ Tools \& Technologies Used

\* \*\*Data Visualization\*\*: Power BI

\* \*\*Dataset Used\*\*: Adult Census Income Dataset
